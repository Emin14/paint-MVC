## Run Locally

Clone the project

```bash
  git clone https://github.com/Emin14/paint-MVC.git
```

Go to the project directory

```bash
  cd paint-MVC
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```


## Features

- Color change
- Selecting brush thickness
- Eraser
- Fill
